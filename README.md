# BatchFileReader


Program to map batch files(.txt) into Java custom objects

<b><i>Currently supports</i></b><br>
1.All Data types<br>
2.Custom Data types<br>
3.List<br>
4.List with custom data types<br>
Features coming:
1.Parallelism Mode 
2.Standard Format Mode
3.Json Mode (currently use batch file)
