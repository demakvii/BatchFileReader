# BatchFileReader

<b><i>Currently supports</i></b><br>
1.All Data types<br>
2.Custom Data types<br>
3.List<br>
4.List with custom data types<br>

<br>2.Standard Format Mode
<br>3.Json Mode (currently use batch file)
